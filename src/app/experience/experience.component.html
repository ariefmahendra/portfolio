Booking Room App

Github

"Booking Room App" is a Go-based Restful API for company room bookings. It utilizes PostgreSQL for the database, JWT tokens for authentication, and follows clean architecture principles with unit testing. The application is containerized using Docker for streamlined deployment on a server.

Mirror and Stationary Cycle (Mirocle)

Github Youtube

Mirocle is a therapeutic device combining mirror therapy with a stationary bike to help stroke patients recover motor function. It leverages the Internet of Things (IoT) to monitor health metrics like heart rate, oxygen saturation, calories burned, pedal revolutions, and exercise duration. This data is accessible through a website using the MQTT protocol (Quality of Service level 0) and can be downloaded in CSV format for further analysis.
